# Cinema

A GUI application to manage movies using Tkinter, SQLite, and Python.

## Features

- Add, view, and manage movies
- Stores data in SQLite
- Scrape movie info from web using BeautifulSoup
- Supports images with Pillow

## Download the repository

### Clone the repository:
```bash
git clone https://gitlab.com/HassanBenbrik/cinema.git
cd cinema
```

### Install dependencies:
```bash
pip install -r requirements.txt
```

or

```bash
pip install .
```

## Lunch the application
```bash
python Movies.py
```
